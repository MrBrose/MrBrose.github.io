FONT INFO - GUIDE ON HOW TO USE SOME COMPLEX FONTS

-- MC Ten Toony --

"~" is a topline for short letters (like a)
"¬" is a topline for loooong letters (like m or w)

-- MC Ten Outline --

Use "{" to start an outline.
Then, use "}" to end an outline.

-- MC Ten Extend --

Lowercase letters have extended bottoms, if they can be placed in a way that makes sense. Otherwise they are just the normal version

-- Any fonts with Alt lowercase or AltLw --

The lowercase versions of some letters will have alternate designs.

-- MC Ten Icons --

Lowercase letters are replaced with icons.
